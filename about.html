Project Info

Project Afterburner (or Afterburner Mod) is a OS mod for ARM platforms made by monkaBlyat. It's made with one goal in mind, speed. "If you paid for the whole SBC, then you should be able to use the whole SBC."


Features:

Each board's distribution is modified differently to the specific needs of the board. In general, all of the versions will preinstall Box86/64, Wine, and retro emulators like RetroArch. GNOME in combination with Wayland is used for the DE, providing the user with a smooth desktop experience. There are also built-in Youtube playback optmizations, allowing you to steam in 4k with zero lag on boards like the RockPi 4.
